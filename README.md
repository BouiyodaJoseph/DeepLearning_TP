# TP - De la Conception au Déploiement de Modèles Deep Learning

Ce projet est une réalisation dans le cadre des Travaux Pratiques de 5ème année Génie Informatique à l'ENSPY.

L'objectif est de couvrir l'ensemble du cycle de vie d'un modèle de Deep Learning, de l'entraînement à la mise en production avec Docker.

## Contenu

- **Partie 1 :** Entraînement d'un réseau de neurones sur le jeu de données MNIST avec Keras/TensorFlow.
- **Partie 2 :** Mise en place du versionnement avec Git, du suivi d'expérimentations avec MLflow et du déploiement via une API Flask conteneurisée avec Docker.
